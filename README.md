# evsrc-cytoscape

Graph correlations between files in git commit history from https://github.com/armish/evsrc/


## Build

To deploy to the dist folder

    gulp

Or un-minified

    gulp devmode


Or in watch mode

    gulp watch

Or in dev watch mode

    gulp watchdev

Requires browserify, uglifyjs
